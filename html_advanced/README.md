Holberton Web Front-End — HTML Advanced

📄 Project Description

This project contains a series of advanced HTML exercises from Holberton School. The goal is to take a beginner developer to a professional level by building semantic, accessible, and maintainable web pages.

It covers:

Semantic HTML5 structure (header, main, section, article, aside, footer)

Text formatting: headings, paragraphs, lists, quotes, code, and typography

Tables with accessibility (scope, caption)

Images, videos, and audio players

SVG icons for social media

Iframes and multimedia integration

Best practices for SEO and accessibility (A11Y)

Each task is an incremental exercise, resulting in fully functional HTML pages.

⚡ Key Features
HTML Structure

Semantic tags for header, main, sections, articles, and footer

Proper heading hierarchy <h1> → <h6>

Paragraphs <p> and lists (ul, ol, dl)

Inline quotes <q> and blockquotes <blockquote>

Tables with accessible headers and captions

Media

Images with alt, width, height, and lazy loading

Video <video> with controls, loop, and poster image

Audio <audio> with controls and fallback text

Iframe integration for embedding external content

Footer & Social Media

Structured footer with address and social links

SVG icons for Facebook, Twitter, Instagram (25px × 25px)

🛠 Technologies Used

HTML5: Semantic structure, multimedia, accessibility

SVG: Social media icons embedded directly

Media: Video and audio players with controls

Responsive Design: Meta viewport, image sizing

Accessibility: alt attributes, semantic markup

🚀 How to Test

Clone the repository:

git clone https://github.com/SaraEstelle/holbertonschool-web_front_end.git

Navigate to the HTML folder:

cd holbertonschool-web_front_end/html_advanced

Open any HTML file in your browser:

open index.html   # MacOS
xdg-open index.html  # Linux

Verify images, videos, audio, and social icons load correctly.
